Argument/Proposal

Source Control Explorer defaulted to the right side so that when switching branches or checking out previous commits, you can easily access the changes.

Make use of Source Control Explorer Pane to:

  • Pending Changes
  • Graph View (DEFAULT VIEW) - Reflect a graph of the branches
    • Graph - Should be interactive to quickly jump to different states of the project
      • Lines reflect branches
      • Nodes on lines show each commit
        • Hover/Click displays the Commit ID with option(s):
          • Jump to - Are you sure? prompt > Puts whole state of project back or forward to how is was during that commit
    • Actions column - displays in Graph View
    • Message - Commit comment
    • Date - of commit
  • Repo View - Reflect a snapshot of files in the repo (Optional, may be overkill)?
    • If so, split pane
    • Hover - Date and commit ID
    • Double click - nothing overkill right now
  • Commit Log View- An alternative view, showing text list of commits with Commit ID (Non-interactive?)
